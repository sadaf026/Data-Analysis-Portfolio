# Retail Sales Analysis

## Overview

This project analyzes a retail sales dataset to identify patterns in revenue, product performance, and sales trends.

## Objectives

* Explore sales distribution
* Identify top-performing products
* Analyze revenue trends
* Visualize relationships between sales variables

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## Key Analysis

The notebook includes data cleaning, exploratory data analysis, correlation analysis, and visualizations to uncover sales insights.

## Files

retail_sales_analysis.ipynb – Main analysis notebook
