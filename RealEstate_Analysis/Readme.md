# Real Estate Market Analysis

## Overview

This project analyzes housing market data to understand property price trends and key factors influencing housing values.

## Objectives

* Analyze price distribution
* Explore relationships between property features
* Identify market trends

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Key Analysis

The notebook explores correlations between house price, area, and other property attributes using visualizations and statistical summaries.
