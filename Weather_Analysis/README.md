# Weather Data Analysis

## Overview

This project analyzes weather data to understand temperature trends and precipitation patterns.

## Objectives

* Study temperature distributions
* Analyze rainfall patterns
* Identify correlations between weather variables

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Key Analysis

The notebook uses visualizations such as histograms, scatter plots, and heatmaps to explore weather trends.
